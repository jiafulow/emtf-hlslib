# emtf-hlslib

This package contains HLS codes for EMTF++. It requires Vivado HLS v2020.1.
It is still under development.

## Getting Started

1. Compile and run:

``` bash
git clone git@github.com:jiafulow/emtf-hlslib.git
cd emtf-hlslib
vivado_hls -f run_hls.tcl
```
